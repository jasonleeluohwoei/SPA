How to run Project
1. Download and Unzip the file on your local system copy SPA .
2. Put SPA folder inside root directory (for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

Database Configuration

Open phpmyadmin
Create Database spa
Import database spa.sql (available SQL File Folder inside zip package)
